# <a name="contacting-item-owners"></a>항목 소유자에게 문의

특정 항목의 소유자에게 문의하려면 항목 세부 정보 페이지로 이동합니다.
왼쪽 메뉴 모음에 소유자에게 문의 링크가 있습니다.
링크를 클릭하면 새 페이지로 이동됩니다.
여기서 메시지를 보낼 수 있습니다.
