# <a name="dsc-resource-script-debugging"></a>DSC 리소스 스크립트 디버깅

WMF 5.0에서는 대상 노드에서 실행되는 DSC 리소스 스크립트 디버깅을 지원합니다.
자세한 내용은 [DSC 리소스 디버그](https://msdn.microsoft.com/powershell/dsc/debugresource)를 참조하세요.
