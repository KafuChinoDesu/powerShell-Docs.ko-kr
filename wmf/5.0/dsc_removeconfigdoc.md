# DSC 문서 제거

구성 문서가 DSC에 제공되면 다양한 단계(보류 중, 현재, 이전)를 거칩니다. Windows PowerShell 4.0에서 새 cmdlet Remove-DscConfigurationDocument를 [KB3000850](https://support.microsoft.com/en-us/kb/3000850)의 일부로 DSC에 추가했습니다.
