# <a name="wow64-support-for-configuration-keyword"></a>WOW64에서 구성 키워드 지원

이제 64비트 컴퓨터의 WOW64에서 구성 키워드가 지원됩니다. 즉, 64비트 컴퓨터에서 실행되는 Windows PowerShell ISE(x86) 같은 32비트 프로세스 내에서 DSC 구성을 정의하고 컴파일할 수 있습니다.
